# Créditos & Inspiração

Este plugin foi **forjado do zero**. As referências abaixo serviram apenas como **norte**
de qualidade e formato — nenhum conteúdo foi copiado; a estrutura, as regras e os textos
das skills, agentes, comando e hooks são originais deste produto.

## Norte de referência
- **Roteiro da Aula 02 (Identidade & Funil Completo)** — inspirou o encadeamento
  identidade → método → página → criativos, o diagnóstico de estágio de consciência, a
  página de 16 elementos e o loop de aprovação humana ("aprovo").
- **Extensão de marketing (sleep_agent_v3)** — norte de organização de agentes e conhecimento
  de copy/funil.

## Ferramentas externas citadas (dependências funcionais, não inspiração)
- **Cloudflare Pages** — host da página publicada, via integração Git (auto-deploy no push).
- **Git** — versionamento da página conectado ao Cloudflare Pages.

## Relação com o RECON
O DEPLOY é o lado **execução** da linha; a **inteligência** (avatar, dossiê, oferta) é do
plugin **RECON**, cujo Offerbook o DEPLOY consome. O DEPLOY nunca re-espiona o concorrente.

## Forjado com
- **Forja de Habilidades** — processo de construção (escopo → componentes → têmpera →
  validação → empacotamento). As três disciplinas (honestidade da copy, loop de aprovação,
  gate de publicação) foram temperadas sob pressão adversarial.
